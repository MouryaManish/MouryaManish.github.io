Project

Detecting schizophrenia through eye movement analysis using artificial neural networks (Deep Learning)

Technology:- Python, NumPY, Deep Learning

Built convolutional and feedforward model with batch normalization and dropouts to classify between healthy people and schizophrenic patients based on saccadic eye movement. This was built on a high performance compute cluster offered by the University of Massachusetts Boston. Achieved an accuracy of 93.87% with 11-fold cross-validation. The project was built by following an online course from Stanford CS231n by Dr. Fei-Fei Li. Many thanks to the Stanford team!
